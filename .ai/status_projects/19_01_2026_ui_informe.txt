## REQUISITO #2: IMPLEMENTAR SUB-VISTAS EN CONTENT_FRAME

### Modificar MainView para soportar sub-vistas:

```python
def show_sub_view(self, sub_view_name: str, sub_view_class):
    """Mostrar sub-vista dentro de la vista actual"""
    # Ocultar vista actual principal
    if self.current_view:
        self.current_view.grid_forget()
    
    # Cache de sub-vistas
    if sub_view_name not in self.sub_views:
        sub_view = sub_view_class(self.content_frame)
        sub_view.grid(row=0, column=0, sticky="nsew")
        self.sub_views[sub_view_name] = sub_view
    
    self.current_view = self.sub_views[sub_view_name]
    self.current_view.grid(row=0, column=0, sticky="nsew")
```

## REQUISITO #4: IMPLEMENTAR HISTORIAL DE NAVEGACIÓN

### Agregar a MainView:

```python
def __init__(self, master, current_user: UserModel):
    # ... existing code ...
    self.navigation_history = []  # Stack de navegación
    self.sub_views = {}  # Cache de sub-vistas

def navigate_to(self, view_name: str, view_class, keep_history=True):
    """Navegar con historial"""
    if keep_history and self.current_view:
        self.navigation_history.append((self.current_view_name, self.current_view))
    
    self.show_view(view_name, view_class)

def go_back(self):
    """Volver a vista anterior"""
    if self.navigation_history:
        prev_name, prev_view = self.navigation_history.pop()
        self.current_view = prev_view
        self.current_view.grid(row=0, column=0, sticky="nsew")
```
