# CONTEXTO DEL PROYECTO – GES

GES es un sistema de gestión escolar offline-first para escuelas locales.

## PRINCIPIOS

* Offline primero
* Simplicidad
* Datos locales
* Historial académico persistente
* Escalable sin reescrituras

## MVP INCLUYE

* Login local
* Registro de estudiantes, docentes y personal
* Matrícula con opción de calendario por tutor
* Registro de notas por trimestre
* Cálculo automático de aprobado/suspenso
* Gráficas básicas de rendimiento
* Importación inicial desde Excel
* Historial académico local por año

## MVP NO INCLUYE

* Licencias online
* Cloud
* Multi-centro
* App web

## USUARIO FINAL

* Secretarios
* Directores
* Personal administrativo
  No técnicos.

## RESTRICCIONES

* Internet limitado
* Hardware modesto
* Código claro y mantenible
